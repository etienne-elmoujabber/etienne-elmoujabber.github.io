HTML Tags

Tags commonly used in the <head> element.

Tags Description and Functionality
<head> The <head> (header) element is used to represent the collection of metadata for the document. Such as linking your CSS stylesheet, Fonts and Script Elements. Usually using a text editor such as VScode, typing "html" and clicking "html:5" will automatically add a common template of <meta> elements used to start coding.
MDN Reference
<meta> The <meta> element represents various kinds of metadata that cannot be expressed using the title, base, link, style and script elements.
MDN Reference
<link> The <link> element is used to link documents to your "HTML" such as Fonts, Folders, CSS and others. When using <link> it is usally followed by the href (href adds a link to a tag) attribute and rel (specifies the relationship) attribute.
I.E. : <link href="RELATIVE PATH TO CSS" rel="stylesheet"> or <link href="FONT LINK" rel="preconnect">.
MDN Reference
<title> The <title> element is your webpage title. Commonly seen on a browser TAB and link path.
MDN Reference
<style> The <style> element allows you to directly style your HTML content using CSS from your HTML document. It is generally better practice to use an external stylesheet but can be handy to provide small examples.
MDN Reference

Tags commonly used in the <body> element.

Tags Description and Functionality
<header> The <header> element is simply your introduction to your website. It may also include a set of navigation links.
MDN Reference
<section> The <section> element is used to represent/seperate a certain section of the HTML file.
MDN Reference
<div> The <div> element is simply used as a container for your HTML <body>. Basically, it helps you divide your HTML into seperate "containers" which makes it easier to read code. Also, it helps you style the content within said "container" without it affecting all of your HTML <body>.
MDN Reference
<h1> <h2> <h3> <h4> <h5> <h6> The <h1>, <h2>, <h3>, <h4>, <h5> and <h6> elements are section headings. Ranking from most important/bigger (1) to least important/smaller (6).
MDN Reference
<p> The <p> element is a paragraph.
MDN Reference
<img> The <img> element is used to add an image to your HTML <body>. When using <img> it is usually followed by src attribute(link to the image) and alt attribute(text description for the visually impaired).
I.E. <img src="LINK TO A WOLF IMAGE" alt="Image of a brown wolf."
MDN Reference
<a> The <a> element is used to add a link to a text. When using <a> it is usually followed by href (href adds a link to a tag).
I.E. <a href="link"> MDN Reference </a> Looking at the page the MDN Reference is a perfect example of this.
MDN Reference
<nav> The <nav> element is used to add navigation links to your website page.
MDN Reference
<ol> The <ol> element is used to create an "Organised List". Not to be confused with the <ul> tag. An "Organised List" is numbered.
MDN Reference
<ul> The <ul> element is used to create an "Unorganised List". Not to be confused with the <ol> tag. An "Unorganised List" is not numbered but bulleted.
MDN Reference
<li> The <li> element is used to list the items in a <ol> tag and <ul>.
MDN Reference

How to create a table in the <body> element.

Tags Description and Functionality
<table> The <table> element is used to create and represent the data in the form of a table.
MDN Reference
<thead> The <thead> element is the block of headers under the <table> element.
MDN Reference
<th> The <th> element defines the header data in the <thead> element.
MDN Reference
<tr> The <tr> element defines a row in a table. It can be used under <thead> and <tbody>.
MDN Reference
<td> The <td> element defines the data in a row or table. It is generally used under the <tbody> element. Not to be mistaken with <th> element.
MDN Reference
<tfoot> The <tfoot> element defines a row summarizing the columns in the table.
MDN Reference

CSS Properties

Properties commonly used to style the <body> using CSS

Properties are used inside your selectors brackets. "{ }".

Properties Description and Functionality
margin The margin property is a shorthand used to define all four sides of the margin area.
"body { margin: auto }"
The "auto" value centers your content using the available horizontal space.
MDN Reference
border The border property defines a border.
MDN Reference
padding The padding property defines the padding area on all four sides around the selectors content.
MDN Reference
height The height property defines the height of the selectors content.
MDN Reference
width The width property defines the width of the selectors content.
MDN Reference
font-family The font-family property defines the font of the contents text.
MDN Reference
color The color property defines the foreground color of the contents text.
MDN Reference
font-weight The font-weight property defines the "boldness" of the font.
MDN Reference
font-size The font-size property defines the size of the font.
MDN Reference
text-align The text-align property sets the horizontal alignment of the text.
MDN Reference
background-image The background-image property sets a background image of the content.
MDN Reference
display The display property sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex.
MDN Reference
box-sizing The box-sizing sets how the total width and height on an element is calculated.
MDN Reference
